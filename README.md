## 1. Sobre o Repositório



Este repositório reúne os **exercícios práticos desenvolvidos durante o Bootcamp JavaScript Full Stack** da [Generation Brasil](https://brazil.generation.org/).

O conteúdo abrange desde os **fundamentos da linguagem TypeScript** até conceitos essenciais para lógica de programação, como:

- Estruturas condicionais
- Laços de repetição
- Manipulação de arrays e collections

Os exemplos foram desenvolvidos com base nas aulas ao vivo (*Live Codes*) e em exercícios práticos voltados para **situações do dia a dia**, facilitando a fixação do aprendizado.

## Requisitos



Para executar os exemplos localmente, você precisará de:

- [Node.js](https://nodejs.org/) **versão 20 ou superior**
- [Git](https://git-scm.com/)
- [Visual Studio Code](https://code.visualstudio.com/) ou editor similar

<br />

## Como Executar os Exemplos



1. Clone o repositório

```bash
git clone https://github.com/outwake/IntroducaoTypescript.git
```

2. Abra a pasta do projeto no **Visual Studio Code**
3. No terminal, navegue até a pasta do exercício desejado
4. Execute o arquivo JavaScript com o comando:

```bash
node nome_do_arquivo.js
```

### Exemplo:

```bash
node HelloWorld.js
```
